## 🎯 5단계 시스템 구성

### ✅ Phase 1: 데이터 파이프라인
**파일**: `data/data_pipeline.py`

**기능**:
- OHLCV 가격 데이터 수집
- 거시 지표 (VIX, 금리, 환율)
- 대안 데이터 (뉴스 감성, SNS 언급)
- 기술적 지표 (RSI, MA, 변동성)
- Z-score 정규화

**출력**: 모든 데이터 통합된 "데이터셋"

---

### ✅ Phase 2: 시장 국면 판단
**파일**: `regime/regime_detector.py`

**기능**:
- 4가지 국면 탐지
  - **BULL** 📈 상승장 (추세 매수)
  - **BEAR** 📉 하락장 (관말/공매도)
  - **SIDEWAYS** 🔄 횡보장 (평균 회귀)
  - **HIGH_VOL** ⚡ 고변동성 (변동성 매도)

**입력**: 20일 가격 데이터
**출력**: 국면 + 설명 + 추천 전략

---

### ✅ Phase 3: 신호 생성 (앙상블)
**파일**: `signals/signal_generator.py`

**3개 AI 에이전트 투표**:

| 에이전트 | 전략 | 판단 기준 |
|----------|------|-----------|
| Agent A | 모멘텀 | MA 차이, RSI 추세 |
| Agent B | 평균 회귀 | RSI 과매수/과매도 |
| Agent C | 이벤트 | 뉴스 감성 지수 |

**앙상블**: 2개 이상 동의 시 매수/매도 결정
**출력**: 최종 신호 (BUY/SELL/HOLD) + 강도

---

### ✅ Phase 4: 리스크 관리
**파일**: `risk/risk_manager.py`

**핵심 기능**:

**1. 변동성 타게팅**
```
투자금 = 기본(10%) × (목표변동성/현재변동성) × 신호강도 × 국면배수
```

**2. 포지션 제한**
- 최소: 1% (총 자본)
- 최대: 30% (리스크 관리)

**3. ATR 기반 손절가**
- BULL: 진입가 - 2.5×ATR
- BEAR: 진입가 - 1.5×ATR
- HIGH_VOL: 진입가 - 3×ATR

**출력**: 투자금액, 손절가, 비중

---

### ✅ Phase 5: 실행
**파일**: `execution/trade_executor.py`

**통합 시스템**:
```python
from execution.trade_executor import QuantSystem

system = QuantSystem(capital=100000000)  # 1억 설정
result = system.analyze("AAPL")  # 분석 실행
```

**순서**:
1. 데이터 수집 → 2. 국면 판단 → 3. 신호 생성 → 4. 리스크 계산 → 5. 실행
