# 파파의 일일 주식 브리핑 시스템

## 🎯 개요
매일 저녁 8시, 확률 70%+ 종목을 자동으로 분석해서 알려주는 시스템

---

## ⏰ 브리핑 시간
- **시간**: 매일 저녁 8시 (20:00 KST)
- **빈도**: 매일 (주말 포함)
- **방식**: 텔레그램으로 목록 전송

---

## 📋 브리핑 내용

```
🎯 파파의 오늘 추천 종목 (2026-02-24)

🥇 1위: META (80%)
   현재가: $133.92
   시그널: RSI 과매도 + 과도한 하락
   → STRONG_BUY

🥇 2위: PYPL (80%)  
   현재가: $122.15
   시그널: RSI 과매도 + 바닥 근접
   → STRONG_BUY

🥉 3위: GOOGL (70%)
   현재가: $130.93
   시그널: RSI 약과매도
   → STRONG_BUY
...

💡 분석 기준:
   • RSI < 35 (과매도)
   • 20일 수익률 < -10%
   • 확률 70% 이상

⚠️ 참고: 실제 매수 전 리스크 관리 확인 필수!
```
