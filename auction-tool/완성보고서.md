## 💡 사용 예시

### 파이썬 코드
```python
# 물건 정보 입력
auction = {
    "address": "서울 마포구",
    "appraisal_price": 500_000_000,  # 감정 5억
    "min_bid_price": 350_000_000,   # 최저 3.5억 (30% 할인)
    "building_type": "아파트",
    "area_pyeong": 30,
    "strategy": "flip"
}

# 등기부 텍스트
document = """
[등기부]
[저당권] 은행, 3억원, 2023년 6월
임차인 없음
"""

# 분석 실행
system = AuctionAnalyzerSystem()
result = system.analyze_auction(auction, document)

# 결과 확인
print(f"수익률: {result['profit_analysis']['profit']['roi']}%")
print(f"위험: {result['rights_analysis']['risk_score']}/100")
print(f"추천: {result['recommendation']}")
```
