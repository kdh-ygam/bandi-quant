# 📖 지지옥션 크롤러 사용법

## 🎯 크롤러가 하는 일
1. Chrome 브라우저 켜기
2. 지지옥션 웹사이트 접속
3. 경매 물건 정보 수집
4. CSV/JSON 파일로 저장

## 📝 코드 설명 (쉽게)

### 클래스 (Class)
```python
class GgiAuctionScraper:
```
→ "지지옥션 크롤러"라는 **도구 상자**

### 초기화 (세팅)
```python
def __init__(self, headless=True):
```
→ Chrome 브라우저 설정
- `headless=True` : 창 안 보이게 (빠름)
- `headless=False` : 창 보이게 (확인용)

### 검색 기능
```python
search_auctions(sido="서울특별시", gugun="", page_limit=3)
```
- `sido` : 시/도 이름
- `gugun` : 구/군 이름
- `page_limit` : 몇 페이지 검색할지

### 저장 기능
```python
save_to_csv()      # 엑셀 파일로 저장
save_to_json()     # JSON 파일로 저장
```
