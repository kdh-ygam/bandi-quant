## 🎯 이 프로젝트는 뭔가요?

**경매 물건을 AI가 자동으로 분석해주는 프로그램**

파파가 경매 물건의 **등기부**를 넣으면:
1. 얼마에 사야 하는지
2. 얼마를 벌 수 있는지
3. 위험은 뭔지
4. 사도 될지 안될지

**한 번에 알려줘요!** 🤖

### 방법 1: 전체 테스트 (추천)
```bash

python3 complete_system.py
```
→ 샘플 분석 2개 보여줌 (좋은 물건 + 나쁜 물건)

---

### 방법 2: 실제 물건 분석
```python
# 파이썬 실행
from complete_system import AuctionAnalyzerSystem

system = AuctionAnalyzerSystem()

# 물건 정보 입력
result = system.analyze_auction(
    {
        "address": "서울 마포구",           # 주소
        "appraisal_price": 500_000_000,     # 감정가 (5억)
        "min_bid_price": 350_000_000,     # 최저입찰가 (3.5억)
        "building_type": "아파트",          # 건물 유형
        "area_pyeong": 30,                  # 평수
        "strategy": "flip"                  # 전략 (flip=매도, rent=임대)
    },
    """
    [등기부정보]
    여기에 등기부 텍스트 붙여넣기
    """
)

# 결과 출력
system.print_report(result)
```

## ⚠️ 주의사항

1. **실제 경매 투자는 본인 책임**
   - 이 프로그램은 "참고용"이에요
   - 최종 결정은 파파가!

2. **등기부 확인 필수**
   - 프로그램이 키워드 찾아주지만
   - 중요한 건 파파가 직접 확인해야 해요

3. **현장 답사 권장**
   - 건물 상태, 누수 등은
   - 직접 가봐야 알 수 있어요
