## 🔄 일일 브리핑 워크플로우

### [Step 1] 📊 데이터 수집 (자동)
- 시간: 미국 장 마감 직후 (매일 06:00 KST)
- 내용:
  - 모든 종목 현재가 조회 (43개)
  - RSI 계산 (14일 기준)
  - 거래량 확인
  - 전일대비 등락률

### [Step 1.5] 📈 차트 생성 (🔥 필수!)
> **⚠️ 중요: 반디퀀트에서 차트는 절대 빠지면 안됨!**
> 파파가 직접 요청한 필수사항. 차트 없으면 브리핑 성립 안됨!

- **정책**: 2026-02-28 ~ 모든 브리핑 차트는 **6개월(120일) 캔들차트**
- 생성 도구: `chart_standard.py`
- 포함 내용:
  - 🕯️ 6개월 OHLCV 캔들 차트 (전체 120일 표시)
  - 🔍 패턴 자동 감지 (망 hammer, 잉걸불, 모닝스타 등)
  - 📊 RSI(14) + 볼린저밴드 + 이동평균선(5/20/60)
  - 📊 거래량 차트
- **금지**: 선형 차트 (Line/Area 차트) 사용 금지
- **전송**: 상위 5개 종목의 차트 이미지를 텔레그램에 필수 첨부!

### [Step 2] 🔍 뉴스 검색 (ddgs)
- 시간: 가격 수집 후 즉시
- 방법:
  ```bash
  ddgs text -q "{종목명} 주가 뉴스" -m 5
  ddgs text -q "{ticker} stock news today" -m 5
  ```
- 저장: `news/{YYYYMMDD}/{ticker}_news.txt`

### [Step 3] 📈 분석 자동화
- 6가지 항목 분석 템플릿 적용:
  1. 💰 투자 현황 (시세/등락)
  2. 🏢 기업 내부 동향 (뉴스 기반)
  3. 📈 영업 전망
  4. 🌍 정세적 흐름
  5. 🎯 매수/매도 등급
  6. 📝 추천 의견

### [Step 4] 🎯 등급 판정 기준

#### 매수 등급
| 등급 | 조건 | 액션 |
|:---|:---|:---|
| 🟢 강력매수 | RSI < 35, 하락 -10%↓ | 즉시 분할 매수 |
| 🟡 매수권유 | RSI 35-45, 저점 확인 | 참여 매수 |
| 🟠 매수대비 | RSI 45-50, 추세 반전 | 관망 후 진입 |

#### 매도 등급
| 등급 | 조건 | 액션 |
|:---|:---|:---|
| 🔴 강력매도 | RSI > 70, 수익 +50%↑ | 즉시 50% 익절 |
| 🟠 매도권유 | RSI 60-70, 수익 +20%↑ | 점진적 익절 |
| 🟡 매도대비 | RSI 60 이상 또는 손절선 | 매도 준비 |

### [Step 5] 🎙️ 샤샤 음성 브리핑
- **조건**: 파파에게 사전 문의
- 생성 항목:
  - 상위 매수 추천 3종목
  - 매도 권고 종목
  - 요약 코멘트
- **확인 후** 텔레그램 전송

### [Step 6] 📱 텔레그램 전송
