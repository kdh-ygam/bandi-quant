# 네이버 공식 API 신청 가이드

## 📝 준비사항

### 필수 항목
- ✅ 네이버 계정 (있으신가요?)
- ✅ **사업자등록증** (개인사업자 OK) - **필수!**
- ✅ 서비스 URL 또는 개발중인 앱 설명

---

## Step 1: 개발자 포털 가입

### 1-1. 개발자 포털 접속
```
https://developers.naver.com/
```

### 1-2. 로그인
- 네이버 아이디로 로그인
- "Developer Console" 클릭

### 1-3. 신청서 작성
```
▶ 프로필 설정
   - 개발자 유형: 개인개발자 (또는 사업자)
   - 연락처: 휴대폰번호
   - 이메일: ...@naver.com
```

---

## Step 2: 애플리케이션 등록

### 2-1. Application 등록
내 애플리케이션 → "애플리케이션 등록" 클릭

### 2-2. 필수 입력
```
애플리케이션 이름: BANDI Quant Stock Analysis
사용 API: 
  ✅ 데이터랩 검색어트렌드 (선택)
  ⬜ 네이버ID로그인
  ⬜ 검색
  ⬜ 지도
  ⬜ 주소검색
  
환경설정:
  PC웹: ⬜ (체크하지 말 것)
  모바일웹: ⬜ (체크하지 말 것)
  
개발환경 URL: http://localhost:8080
서비스 URL: https://github.com/bandi/investment (예시)

사업자등록번호: [개인사업자 번호 입력]
```

**주의**: 주식 가격 API는 별도 **증권 데이터 API** 신청 필요!

---

## Step 3: 증권/금융 데이터 API 별도 신청

### ⚠️ 중요: 개발자 포털 API ≠ 주식가격 API

네이버 개발자 포털에는 **실시간 주가 API가 없습니다.**

### 실제 금융 데이터 제공처:

#### 옵션 1: **금융위원회 API** (공공데이터포털)
```
1. 공공데이터포털 (data.go.kr) 가입
2. "한국거래표 API" 검색
3. API 신청
4. 승인 대기 (1-3일)
```

#### 옵션 2: **한국투자증권 API (KIS)** ⭐추천
```
URL: https://api.hinvest.com/

신청 절차:
1. 한국투자증권 계좌 개설 (필수!)
2. 모의투자 계좌 생성 (무료)
3. API 키 발급 신청
4. 승인 (즉시~3일)
```

**장점**: 실시간 주가, 무료, 공식
**단점**: 계좌 필요, 복잡함

#### 옵션 3: **네이버 금융 데이터 (비공식)**
```
⚠️ 주의: 네이버 공식 API가 아닙니다
네이버 금융 홈페이지 스크래핑은 서비스약관 위반 가능성
```

---

## 🎯 반디 추천: 가장 현실적인 방법

### 최선책: **한국투자증권 KIS API** (⭐추천)

**특징:**
- ✅ 무료로 실시간 주가 제공
- ✅ REST API (JSON)
- ✅ KOSPI/KOSDAQ/KONEX 전부 지원
- ✅ OHLCV + 호가 + 체결 제공

**필요 조건:**
- 한국투자증권 계좌 (실제 또는 모의)
- PC + 공동인증서 (또는 간편인증)

**대략 소요 시간:**
- 가입: 30분
- API 승인: 즉시 ~ 3영업일

### 차선책: **공공데이터포털 한국거래소 API**

**특징:**
- ✅ 완전 무료
- ✅ 공식 데이터
- ❌ 지연시간 있음 (15-30분)
- ❌ 복잡한 인증 절차

## ❌ 안타깝게도...

### 네이버에 없는 것
- 일반 개발자용 실시간 주가 API
- 무료 OHLCV 데이터 API
- REST API 형태의 금융 데이터

### 네이버 검색 API로 가능한 것
- ✅ 검색어 트렌드 (종목 관련 검색량)
- ✅ 뉴스 검색 결과
- ✅ 블로그/카페 내용
- ❌ 실시간 주가 ❌
